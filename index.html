My Projects

  • Rob's Journey

    I was a playtester, artist, gameplay designer, and programmer for the web based puzzle game Rob's Journey. This game was made as a Capstone project in a group of five. The game was designed to teach players about the Psyche mission as well as the basics of computer logic gates.


  • Video Game Database

    I was one of three developers who made a PostgreSQL database containing information about various video games. The database used the IGDB API and the APIcalypse wrapper to add information to a local database. An ASP.NET form was developed to allow users to search the database using a GUI.


  • Restful Microservices

    I made multiple services that called government APIs in order to display information about crime rates, as well as the sunshine index, of user given cities and states. After creating these two services, they were combined with two other services designed by another programmer. Together we made a six page website hosted on ASU servers that had an index table and a page for each service. (These services are no longer hosted as the server was closed).
